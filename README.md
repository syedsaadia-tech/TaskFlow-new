# TaskFlow

TaskFlow is a task management web application that helps users organize and manage tasks efficiently with a simple and responsive interface.

## Features

* User Authentication
* Create, Update & Delete Tasks
* Task Status Management
* Responsive UI
* REST API Integration

## Technologies Used

### Frontend

* HTML
* CSS
* JavaScript
* React.js

### Backend

* Python
* Django
* Django REST Framework

### Database

* SQLite / MySQL

### Tools

* Git
* GitHub
* VS Code
* Postman

## Run Project

```bash
# Backend
python manage.py runserver

# Frontend
npm start
```
